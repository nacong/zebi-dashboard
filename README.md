# 학생회를 위한, 제비 대시보드
## 핵심 기능

**데이터 테이블 & 페이지네이션**
- 서버 사이드 페이지네이션 구현
- 페이지당 항목 수 조절 (10, 20, 50개)
- 페이지 번호 네비게이션

**검색 & 필터링**
- 실시간 검색 기능 (디바운싱 적용)
- 다중 필터 옵션 (상태, 카테고리 등)
- 날짜 범위 필터링

**정렬 기능**
- 컬럼별 오름차순/내림차순 정렬
- 정렬 상태 시각적 표시

**CRUD 작업**
- 데이터 생성, 조회, 수정, 삭제
- 모달 기반 폼 인터페이스
- 폼 유효성 검증

**UI/UX**
- 반응형 레이아웃 (모바일, 태블릿, 데스크톱)
- 로딩 스피너 및 스켈레톤 UI

**상태 관리**
- React 상태 관리 (useState, useEffect)
- API 호출 상태 관리
- 에러 핸들링
